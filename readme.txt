## 简介
Photo Frame 是一个可以对图片添加自定义拍立得风格相框并增加溯源水印的工具。

## 功能
- 添加拍立得风格相框
- 增加溯源水印
- 支持多种图片格式

## 安装
请确保您的环境中已安装 [Node.js](https://nodejs.org/) 和 [npm](https://www.npmjs.com/)。

1. 克隆此仓库：
    ```bash
    git clone https://github.com/l6565kj/photo-frame.git
    ```
2. 进入项目目录：
    ```bash
    cd photo-frame
    ```
3. 安装依赖：
    ```bash
    npm install
    ```

## 使用
1. 启动项目：
    ```bash
    npm start
    ```
2. 打开浏览器并访问 `http://localhost:3000` 开始使用。
